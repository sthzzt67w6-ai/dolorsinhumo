```
dolorsinhumo/
```

## Deployment

El sitio está publicado en **Netlify** y el dominio apunta desde **GoDaddy**.

Para publicar cambios:
1. Edita `index.html`
2. Haz commit y push a `main`
3. Netlify despliega automáticamente en ~30 segundos
